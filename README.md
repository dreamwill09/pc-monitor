# 🖥️ PC Status Monitoring System
파이썬을 이용한 실시간 PC 상태 모니터링 대시보드입니다.

## ✨ 주요 기능
- **실시간 리소스 그래프**: CPU, RAM 사용량 시각화
- **상세 디스크 정보**: C/D 드라이브 용량 및 잔여량 표시
- **네트워크 모니터링**: 실시간 업로드/다운로드 속도(MB/s)
- **프로세스 감시**: CPU 점유율 Top 3 프로세스 표시
- **자동 초기화**: 화면 가독성을 위해 10개 데이터마다 목록 초기화

## 🛠️ 설치 및 실행 방법
1. 서버 실행: `python server.py`
2. 클라이언트 실행: `python client.py`
